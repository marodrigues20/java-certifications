14. Which locations require you to be careful when working with sensitive data to ensure it doesn't 
    leak? (Choose all that apply.)

A. Comments <br>
B. Exception stack traces <br>
C. Log files <br>
D. System.out <br>
E. Variable names <br>
F. None of the above <br>


Answer: B; C; D.


- Any information the user can see requires care. 
- Option B, C, and D are correct for this reason. Comments and variables names are part of the program, not the data it
  handles, making options A and E incorrect.
