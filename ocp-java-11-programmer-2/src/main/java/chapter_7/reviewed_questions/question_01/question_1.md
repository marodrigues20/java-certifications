1. Given an instance of a *Stream s* and *Collection c*, which are valid  ways of creating a parallel stream?

A. new ParallelStream(s) <br>
B. c.parallel()          <br>   
C. s.parallelStream()    <br>
D. c.parallelStream()    <br>
E. new ParallelStream(c)  <br>
F. s.parallel()


## Correct Answers: D; F
